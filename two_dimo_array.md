```python
python中如何用列表推导式生成二维数组？
比如 生成array【10】【5】
```


```python
array = [[j for j in range(5)]for i in range(10)]
print(array)
```

```python
如何把原有的数组转换成array[5][10]
```


```python
array_after = [[array[j][i] for j in range(10)]for i in range(5)]
print(array_after)
```
